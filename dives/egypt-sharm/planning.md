## Split-week schedule (work in the middle)

**Default pattern for this trip and future dive holidays:** five **dive mornings** spread across the week with **two full work days** (remote) between dive blocks — not a single Mon–Fri dive block like Malta Jul 2026.

| Day | Activity | Dives |
|-----|----------|-------|
| **Arrive** (e.g. Fri) | Travel — **no diving**; kit fit / paperwork if offered | — |
| **Sat** | **Dive** — 2-tank morning | 2 |
| **Sun** | **Dive** — 2-tank morning | 2 |
| **Mon–Tue** | **Remote work** — no diving | — |
| **Wed** | **Dive** — 2-tank morning | 2 |
| **Thu** | **Dive** — 2-tank morning | 2 |
| **Fri** | **Dive** — 2-tank morning (**last dives**) | 2 |
| **Sat** | **No diving** — **24 h no-fly** after Fri dives; fly home when safe | — |

**Example wall calendar (adjust to flights):** Fri arrive → Sat–Sun dive → Mon–Tue work → Wed–Fri dive → Sat fly (~8 nights Fri→Sat).

### Why this changes centre choice (vs Malta Seashell)

| Malta (Jul 2026) | Egypt (Sep 2026) |
|------------------|------------------|
| **Best of 3 Islands** — named mix, **5 consecutive** dive days (7–11 Jul) | Need **flexible 10-dive pack** valid across **Sat–Sun gap Wed–Fri** |
| Site variety via **package + email** | Same **written day-by-day site plan** (Thistlegorm, Ras Mohammed, Tiran, locals) — **not** open “10 credits” only |
| Consecutive mornings back ~13:30 | Work days need **reliable return by early afternoon** on dive days; **kit storage** at centre between Sun and Wed |

### Booking requirements (all centres)

Confirm in writing before deposit:

1. **10 dives on 5 named calendar days** — **not** required to be consecutive; validity / expiry across the Mon–Tue gap.  
2. **Draft site plan** per day (no repeat sites unless weather); Thistlegorm on which day (often full-day, early start).  
3. **Back by ~13:30** on boat days where possible (work calls Mon–Tue are off, but same rhythm helps).  
4. **Kit storage** at centre between dive blocks (Sat–Sun then Wed–Fri).  
5. **Own dive computer** (see below) — quote **BCD, reg, wetsuit, weights** only.

### Reusable template (future trips)

When planning any dive-focused trip, assume:

- **5 dive mornings × 2 tanks = 10 dives** (or agreed count).  
- **2 non-dive days mid-trip** for remote work (typically **Mon–Tue** if diving **Sat–Sun** then **Wed–Fri**).  
- **Longer stay** than a consecutive-dive week (~**+2 nights** vs “7 nights, 5 dive days in a row”).  
- Book **itinerary + flexible dates**, not “5 consecutive diving days” only.

## Conditions — late September (expect, not guarantee)

**End of September** is a strong shoulder window: warm sea without peak-summer air stress; fewer boats than Aug.

| | Typical late Sep |
|--|------------------|
| **Air (day)** | ~**30–34°C** — hot; easier than Jul–Aug |
| **Air (evening)** | ~**24–27°C** — OK for Mon–Tue remote work on balcony / AC |
| **Water** | ~**27–28°C** — **3 mm** shorty/full or rash guard; well above your &lt;22°C floor |
| **Visibility** | Often **20–30 m+**; can be superb day to day |
| **Rain** | Essentially none |
| **Wind / sea** | Usually calm; **Ras Mohammed & Tiran** can cancel if wind picks up — need written **weather fallbacks** |
| **Rhythm** | **Morning boat dives**; afternoons for shade / work (Mon–Tue off diving helps surface interval) |

**Kit:** Own **dive computer** + rented BCD/reg/wsuit; **Nitrox** useful on Thistlegorm and deeper walls after Malta cert.

## Sharm resort week vs future liveaboard

### What overlaps?

| Site / area | This Sharm week | Typical **Red Sea liveaboard** (Brothers / Daedalus / Elphinstone) |
|-------------|-----------------|----------------------------------------------------------------------|
| **Thistlegorm** | Yes (planned Thu) | **Often yes** on “north + wrecks” itineraries — **repeat possible** |
| **Ras Mohammed** (Shark & Yolanda) | Yes (Wed) | **Sometimes** on north routes — overlap |
| **Straits of Tiran** | Yes (Fri) | **Sometimes** — overlap |
| **Sharm local reefs** (Garden, Temple, etc.) | Yes (Sat–Sun) | **Rarely** — liveaboards leave harbour for offshore reefs |
| **Brothers / Daedalus / Elphinstone** | No | **Core** liveaboard experience — sharks, current, blue water |
| **St John’s / Fury Shoal / south** | No | **Different** Egypt liveaboard routes entirely |

So: **partial overlap** on northern classics (especially **Thistlegorm**); **not** a duplicate of a **Brothers-style** liveaboard. Your bucket list treats them as **separate items** (`Thistlegorm` vs `Red Sea Liveaboard` in `bucket-list-readiness.md`).

### Different liveaboards elsewhere

**Maldives, Raja Ampat, Komodo, Palau, Galapagos**, etc. cover **entirely different** geography — zero substitution for Sharm. You can do **this trip** and later liveaboards that are **other countries** or **other Red Sea routes**.

### Skip Sharm and wait for liveaboard?

**Usually not worth skipping** if Sep 2026 is still on the table:

| Do **Sharm resort week** | Wait for **liveaboard only** |
|--------------------------|------------------------------|
| **Europe-equivalent** flight (~5.5 h) — not your annual **long-haul** slot | Brothers trips often **7+ days**, pricier, harder to insert **Mon–Tue work** |
| **~35 dives** post-Malta fits **Thistlegorm** readiness (**25–35** in bucket list) | **Red Sea liveaboard** readiness target is **~30–40 dives** + stronger **current / SMB / negative entry** — more realistic **after** this trip |
| Ticks **Thistlegorm** and **boat routine** before harder pelagic trips | “North + wreck” liveaboard **repeats** some sites but adds **Brothers** — different main payoff |
| **Split-week** pattern (work mid-trip) fits **hotel-based** Sharm, not most liveaboards | One liveaboard ≠ all of Egypt; south/St John’s still untouched |

### Decision — proceed with this plan

| Now (2026) | Later (not before ~2028) |
|------------|---------------------------|
| **Sharm resort week** — northern classics + **Thistlegorm**, split-week + work | **Red Sea liveaboard** — **Brothers / Daedalus / Elphinstone** style (not another Naama Bay 10-pack) |
| One **Red Sea** hit this year, then **pause 1–2 years** on Egypt / Red Sea returns | After pause, when dive count + **current / SMB** evidence fit `bucket-list-readiness.md` |

**Mild repeat risk:** Thistlegorm on a future liveaboard is optional — you can skip it that week.

### After this trip — regional sequencing

- **2026:** Malta (Jul) + **Sharm (Sep)** — Red Sea “northern resort” box ticked.  
- **2027:** Prefer **other regions** — e.g. **Caribbean** and **Asia** (bucket-list progression there); **no second Red Sea trip** in 2027 unless you explicitly reopen.  
- **~2028+:** **Brothers-type** Red Sea liveaboard when ready — does not compete with 2027 Caribbean/Asia focus.

Captured in **`trip-preferences.md`** (`destinations_deferred`, `region_focus_2027`).
