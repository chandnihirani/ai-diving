# Diving Project — my dive log + “feedback loop” for getting better

## ELI5 (what this is)
This is my personal system for **storing every dive** and then using that history to give me **interactive feedback** so I improve as a diver.

Think of it like: **“Strava for my scuba learning”** — except it’s built from simple files, so it’s searchable, consistent, and grows with me.

---

## What it does (in human terms)
After each dive, I capture what happened (depth/time/conditions) *and* what I learned (buoyancy, current comfort, mistakes, wins).
Then I can ask questions like:

- **Where am I improving?** (buoyancy, trim, breathing, navigation, boat routine)
- **What are my gaps?** (night dives, stronger current, open-water SMB reps, negative entries, etc.)
- **What should I do next?** (skills to practice, drills, course choices)
- **Where should I dive next?** (places that match my current level *and* fill the next gap)
- **Am I ready for bucket‑list trips yet?** (Maldives / Raja Ampat / Komodo style diving)

The point isn’t just “logging dives” — it’s turning dives into a **repeatable learning loop**.

---

## How I use it (what happens after a dive)
- **Capture**: I drop in my dive info (from the PADI app / screenshots / quick notes).
- **Reflect**: I add what felt easy/hard and what I want to improve.
- **Debrief (optional)**: I do a short Q&A-style debrief on one dive to pull out lessons and next steps.
- **Update readiness**: as new evidence accumulates, the “bucket‑list readiness” view stays current.

---

## Why this also helps me learn tech
I’m using a real hobby as a real dataset. That lets me practice “tech thinking” without it being abstract:

- **Data structure**: consistent fields for every dive (so the history is comparable)
- **Automation**: scripts to organize media and reduce manual admin
- **Analysis mindset**: evidence‑based progress (not vibes), trend spotting, and gap detection
- **Product thinking**: turning messy real life into a system that’s actually useful

So it’s simultaneously: **a better dive log** and **a sandbox for building practical tech skills**.
